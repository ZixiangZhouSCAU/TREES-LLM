# 碳储量计算规范

> 基于 IPCC 碳汇估算标准和中国林业行业标准

## 碳储量估算基本方法

碳储量 = 生物量 × 含碳率

其中：
- 生物量（B）：单位 kg，通过异速生长方程或材积法计算
- 含碳率（CF）：通用值 0.50（针叶树 0.51，阔叶树 0.49）

## IPCC 碳汇估算标准（2006 Guidelines）

### 生物量扩展因子法（BEF法）

AGB = V × WD × BEF

其中：
- AGB：地上生物量 (kg)
- V：树干材积 (m³)
- WD：基本密度 (kg/m³)
- BEF：生物量扩展因子（一般 1.2-2.0）

### 含碳率参考值

| 树种组 | 含碳率 |
|--------|--------|
| 杉木 | 0.520 |
| 马尾松 | 0.510 |
| 阔叶树 | 0.490 |
| 桉树 | 0.480 |
| 通用 | 0.500 |

## 材积-密度法（林业标准方法）

V = Σ(分段圆台体积)
AGB = V × BD
C = AGB × CF

其中：
- V：树干材积 (m³)
- BD：基本密度（阔叶树参考值 0.45-0.55 g/cm³）
- CF：含碳率 = 0.5

## 点云体积法

V_stem = 体素化树干点云体积
V_crown = 凸包/体素化树冠体积
AGB = V_stem × BD_stem + V_crown × BD_crown
C = AGB × 0.5
